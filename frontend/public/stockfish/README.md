Положите сюда **Lichess Stockfish WASM**:

- `stockfish.js`
- `stockfish.wasm` (и прочие файлы, которые тянет билд)

Готовый single-threaded бандл можно взять из репозитория Lichess:
https://github.com/lichess-org/stockfish.wasm

Если файлов нет, клиент автоматически переключится на упрощённый движок на `chess.js` (случайный легальный ход).
